App.js 🔁 ¿Y qué hace esto? Cada vez que se monta el componente App (es decir, cuando inicia tu app), va a revisar si hay un token válido guardado en localStorage. Si lo encuentra, despacha LOGIN_SUCCESS con el token y rol, y tu estado de Redux quedará autenticado. Si no hay token, se despacha logout() (opcional, pero útil para limpiar el estado por si quedó algo viejo).
